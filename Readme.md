Дискреционная модель управления доступом (DAC) — это модель безопасности, в которой владельцы ресурсов (например, файлов, папок, таблиц в базе данных) самостоятельно определяют, кто может получать к ним доступ и какие действия (чтение, запись, выполнение) им разрешены

Мандатная модель (MAC) — это принудительная модель управления доступом, которая ограничивает доступ субъектов (пользователей) к объектам (данным) на основе предопределенных администратором меток конфиденциальности и уровня допуска

┌─────────────────────────┬─────────────────────────────────────┬──────────────────────────────────────────────┐
│ Характеристика          │ DAC (Дискреционная)                 │ MAC (Мандатная)                              │
├─────────────────────────┼─────────────────────────────────────┼──────────────────────────────────────────────┤
│ Кто управляет доступом? │ Владелец объекта                    │ Система / администратор (политика)           │
│ Гибкость                │ Высокая                             │ Низкая                                       │
│ Безопасность            │ Ниже (зависит от пользователя)      │ Выше (принудительный контроль)               │
│ Примеры                 │ Windows NTFS, Linux (chmod, ACL)    │ SELinux, Trusted Solaris, военные системы    │
│ Передача прав           │ Разрешена (по усмотрению владельца) │ Запрещена или строго ограничена              │
│ Подходит для            │ Корпоративных, домашних систем      │ Госбезопасности, критической инфраструктуры  │
└─────────────────────────┴─────────────────────────────────────┴──────────────────────────────────────────────┘
